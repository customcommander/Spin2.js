.crumb1: last panel, hidden

use it when the corresponding panel is the last one and is not currently visible.

.crumb2: hidden

use it when the corresponding panel is not visible and its next sibling isn't neither.

.crumb3: hidden

use it when the corresponding panel is not visible but its next sibling is.

.crumb4: last panel, visible

use it when the corresponding panel is the last one and is visible.

.crumb5: visible

use it when the corresponding panel is visible and so is its next sibling.

.crumb6: visible

use it when the corresponding panel is visible and its next sibling isn't.
